## 🚀 Быстрая установка

### Способ 1: Автоустановщик (Рекомендуется)

1. Скачайте файл **`UZapret_Setup.bat`** (кнопка выше)
2. **Запустите от имени Администратора** (ПКМ → "Запуск от имени администратора")
3. Дождитесь завершения установки
4. На рабочем столе появится ярлык **UZapret**
5. Запустите программу

### Способ 2: Ручная установка

1. Скачайте **`UZapret.exe`**
2. Создайте папку `C:\Users\Ваше_Имя\Documents\UZapret`
3. Поместите `UZapret.exe` в эту папку
4. Создайте ярлык на рабочем столе (по желанию)
5. Запустите от имени администратора

## 🎯 Как пользоваться

1. **Запустите программу от имени администратора**
2. Выберите сервисы, которые хотите разблокировать (Discord, Telegram)
3. Нажмите **"Запустить обходы"**
4. Готово! Сервисы должны работать
